1. Definitions

HTML

It stands for Hyper text markup language. It is the set of symbols or codes used for world wide web browsers.

CSS

CSS stands for cascading style sheet. It is the language which describe how the html element should be displayed in the screen it is used to define styles, tables color and the other aspect of the web page.

JAVASCRIPT

It is the scripting language used in the web. It is used to enhance the html pages and is used to embed the html code. It is the high-level, dynamic and interpreted programming language.

HTTP

It stands for hyper text transfer protocol. It is the standard protocol for transferring the hyper text document into the world wide web. It is the foundation of data communication for the world wide web.

HTML 5

Hyper text markup language version 5 is the language for the structure and presentation of world wide web content. It creates better semantic markup. It allows the creation of more robust web applications.

DYNAMIC HTML

Dynamic HTML is the collective term for combination of new hyper text markup language(HTML)tags and option to create world wide web. It allows user to create highly animated and interactive websites. It provides dynamic elements like drop and drag features, drop down menu and others web page interface.
